# Getting MON Tokens

To use Area51 Bot, you'll need MON tokens for gas fees and trading. Since we're on testnet, these tokens are free and available from faucets.

## What are MON Tokens?

MON is the native token of the Monad blockchain. You need MON for:

* **Gas fees** - To pay for transaction costs
* **Trading** - As a base currency for token swaps
* **Liquidity** - To provide liquidity in pools

## Getting Testnet MON

### Method 1: Official Monad Faucet

1. **Visit**: [faucet.monad.xyz](https://faucet.monad.xyz)
2. **Connect wallet** or enter your Monad address
3. **Request tokens**&#x20;
4. **Wait** for tokens to arrive (usually instant)



### Method 2: Community Faucets

1. **Join Monad Discord**: [discord.gg/monad](https://discord.gg/monad)
2. **Go to #faucet channel**
3. **Use bot commands** to request tokens
4. **Follow community guidelines**

**Benefits**:

* Higher daily limits
* Community support
* Multiple token types available
