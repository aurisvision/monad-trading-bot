# Getting MON Tokens

To use Area51 Bot, you'll need MON tokens for gas fees and trading. Since we're on testnet, these tokens are free and available from faucets.

## What are MON Tokens?

MON is the native token of the Monad blockchain. You need MON for:
- **Gas fees** - To pay for transaction costs
- **Trading** - As a base currency for token swaps
- **Liquidity** - To provide liquidity in pools

## Getting Testnet MON

### Method 1: Official Monad Faucet

1. **Visit**: [faucet.monad.xyz](https://faucet.monad.xyz)
2. **Connect wallet** or enter your Monad address
3. **Request tokens** (usually 1-5 MON per request)
4. **Wait** for tokens to arrive (usually instant)

**Limits**: 
- 5 MON per day per address
- 24-hour cooldown between requests

### Method 2: Community Faucets

1. **Join Monad Discord**: [discord.gg/monad](https://discord.gg/monad)
2. **Go to #faucet channel**
3. **Use bot commands** to request tokens
4. **Follow community guidelines**

**Benefits**:
- Higher daily limits
- Community support
- Multiple token types available

### Method 3: Area51 Community

1. **Join our group**: [@Area51Community](https://t.me/Area51Community)
2. **Ask for help** - Community members often share
3. **Follow group rules** about faucet requests

## Getting Your Monad Address

### From Area51 Bot
1. Open Area51 Bot in Telegram
2. Click **Wallet** button
3. Your address is displayed at the top
4. Copy and use for faucet requests

### From External Wallet
1. Add Monad network to your wallet (see Network Info)
2. Create or import a wallet
3. Copy your Monad address
4. Use for faucet requests

## Troubleshooting

### Faucet Not Working
- **Check cooldown** - Wait 24 hours between requests
- **Verify address** - Ensure you're using correct Monad address
- **Try different faucets** - Multiple sources available

### Tokens Not Arriving
- **Wait a few minutes** - Network can be busy
- **Check block explorer** - Verify transaction status
- **Refresh wallet** - Sometimes display lags behind

### Need More Tokens
- **Use multiple faucets** - Each has separate limits
- **Ask community** - Others may help with small amounts
- **Be patient** - Daily limits reset every 24 hours
