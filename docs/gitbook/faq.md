### How do I create a wallet?

**Step-by-Step Wallet Creation:**

1. **Initialize Bot**
   - Send `/start` to Area51 Bot
   - Access the main menu

2. **Wallet Setup**
   - Select "Setup Wallet" or "Create Wallet"
   - Choose "Generate New Wallet"

3. **Seed Phrase Generation**
   - Bot generates your unique 12-word seed phrase

### How is profit/loss calculated?

**P&L Calculation Method:**

**Basic Formula:**
- **Current Value** = Current Price × Token Quantity
- **Original Value** = Purchase Price × Token Quantity
- **P&L Amount** = Current Value - Original Value
- **P&L Percentage** = (P&L Amount / Original Value) × 100

**Display Indicators:**
- **Green Values** - Profitable positions (gains)
- **Red Values** - Loss positions (losses)
- **Percentage Change** - Relative performance
- **Absolute Change** - MON/USD amount gained/lost

**Factors Considered:**
- **Average Buy Price** - For multiple purchases
- **Current Market Price** - Real-time token value
- **Gas Fees** - Not included in P&L calculation
- **Time-Weighted** - Recent purchases weighted appropriately
