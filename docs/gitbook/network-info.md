# Network Information

This page contains the technical details you need to connect to the Monad network and use Area51 Bot effectively.

## Monad Testnet Details

### Network Configuration

- **Network Name**: Monad Testnet
- **Chain ID**: `41454` (0xa1de in hex)
- **Currency Symbol**: MON
- **RPC URL**: `https://testnet-rpc.monad.xyz`
- **Block Explorer**: [Monad Explorer](https://testnet.monadexplorer.com)

### Adding Monad to Your Wallet

If you want to use external wallets with Monad, add these details:

```
Network Name: Monad Testnet
RPC URL: https://testnet-rpc.monad.xyz
Chain ID: 41454
Currency Symbol: MON
Block Explorer: https://testnet.monadexplorer.com
```

## Troubleshooting

### Common Issues

**"Transaction Failed"**
- Check you have enough MON for gas
- Verify network connection
- Try increasing gas limit

**"Network Not Found"**
- Verify RPC URL is correct
- Check Chain ID (41454)
- Ensure your wallet supports custom networks

**"Insufficient Balance"**
- Get more testnet MON from faucets
- Wait for faucet cooldown period
- Check transaction history for pending txs
