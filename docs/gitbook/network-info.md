# Network Information

This page contains the technical details you need to connect to the Monad network and use Area51 Bot effectively.

## Monad Testnet Details

### Network Configuration

* **Network Name**: Monad Testnet
* **Chain ID**: `10143`&#x20;
* **Currency Symbol**: MON
* **RPC URL**: `https://testnet-rpc.monad.xyz`
* **Block Explorer**: [Monad Explorer](https://testnet.monadexplorer.com)

### Adding Monad to Your Wallet

If you want to use external wallets with Monad, add these details:

```
Network Name: Monad Testnet
RPC URL: https://testnet-rpc.monad.xyz
Chain ID: 10143
Currency Symbol: MON
Block Explorer: https://testnet.monadexplorer.com
```

## Important Addresses

### Native Token

* **MON Token**: Native gas token of Monad network
* **Decimals**: 18
* **Used for**: Gas fees, trading pairs, staking

### DEX Integration

* **Monorail**: Primary DEX aggregator used by Area51 Bot
* **Supported DEXs**: Multiple decentralized exchanges on Monad
* **Liquidity Sources**: Aggregated from various protocols

## Gas and Fees

### Gas Prices

* **Standard**: 50 Gwei&#x20;

### Transaction Costs

* **Simple transfer**: \~0.0001 MON
* **Token swap**: \~0.001-0.005 MON
